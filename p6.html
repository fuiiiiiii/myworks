REMEMBERING MONET
responsive web design + coding
To pay greatest respect to one of my favorite artist Claude Oscar Monet, I chose to create this informational website.
I was lucky enough to see Monet’s real paintings at de Young Museum in San Francisco during Spring 2018.

Project Goal.

The goal of this project is to learn the process of designing a responsive website and to understand the front-end web development process. View

Color Inspiration.

Since Monet is a master in using colors, my primary color palette was inspired by one of his famous oil painting of water lilies. I used Pantone app to extract colors from this painting and applied 50% opacity to them then use as my primary colors for the website.
This website is designed in light colors because the main purpose of this website is to showcase Monet’s masterpieces, hence adding any contrasting colors might takeaway audience’s attention, and that is not ideal for the main purpose of making this website.
